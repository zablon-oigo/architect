## Tech Stack

**Client:** React, TailwindCSS

Clone the project

```bash
  git clone git@github.com:zablon-oigo/architect.git
```

Go to the project directory

```bash
  cd architect
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
